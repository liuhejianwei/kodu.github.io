Kodu的下载与安装—Kodu少儿编程第二天

本文介绍Kodu的下载与安装。

第一步：点击Get Kodu进入下载地址。

下载地址：

https://www.microsoft.com/en-us/download/details.aspx?id=10056

点击DownLoad进行下载。

有两个包可供选择，任选一个进行安装。

第二步：下载文件双击进行安装。

选择您使用的语言：

接受协议：

选择安装目录：

至此，Kodu程序就安装好了。您就可以打开桌面上Kodu图标进入Kodu的开发界面。

Kodu支持的操作系统 ：

Windows 10, Windows 7, Windows 8, Windows Vista, Windows XP

Kodu开发软件是一个很简单易操作的软件，我们来了解一下他的基本功能。

打开Kodu程序，进入软件界面。

新闻：

Kodu的一些新闻。

Sign In:

Kodu的用户系统。Creator是用户名，Pin可以理解为密码。不用注册，直接输入用户名和密码就能进入，这个账户就是你的专属账户。

Keep me signed in when Kodu exits 如果选中，下次登录的时候还进入此账户。

如果不填写用户名和密码，则用系统默认的Guest用户进入，开始学习时不用填写用户名和密码，直接进入即可。进入后如下图：

新闻：

是Kodu官方的一些新闻，组织活动，更新日志等。点击新闻下面的更多，可以看到更多的新闻。可以看到，现在的新闻是Meet and Code的活动。下面是Kodu发布的日志。最新版的2019年6月12日发布的1.5.47.0。

一些选项配置。
