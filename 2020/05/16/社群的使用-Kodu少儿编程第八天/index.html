社群的使用---Kodu少儿编程第八天

2020-05-16 /
  • 分享到社群
  • 社群
 / 点击

社群是Kodu的一个交流平台，用户可以把自己开发的程序上传，也可以下载别人开发的程序。

打开Kodu，点击社群，如下图：

重点说明的是：社群功能需要联网。社群界面如下：

我的世界：我上传的程序和我下载的程序

课程：如果上传的时候选择课程标签，会出现在这里

范例：如果上传的时候没选择课程标签，会出现在这里

全部：包括上面三个

搜索框：在社群里搜索程序，是按作者名称、标题和标签进行搜索

排序：把搜索出来的结果进行排序，可以按日期，创造者，标题，排名分别排序，效果如下图:

我上传了一个程序，可以搜索“社群测试2020”，效果如下：

“社群测试2020”是名称，”v01”是版本号，2020/4/9 15：02是上传日期，“多多”是上传用户，向下的箭号右边的数字是下载次数，“动作，冒险”是标签，“这是说明”是说明。

鼠标左键单击这个程序，效果如下：

点击下载就可以把这个程序下载到本地

点击报告，效果如下：

你要回报此阶层为有冒犯意味的阶层吗？他在英文版里是“do you want to report this level as offensive”，翻译为“你是否举报违规”更合理，可见老外的翻译水平好差。

点击下载，右上角出现小箭头的时候，就可以开始玩了。因为社群的服务器在国外，经常有下载不成功的情况，这时需要重新下载，可见老外的服务器也不好。如下图:

社群上的所有程序都是开源的，我们不仅可以下载，还可以任意编辑和重新上传。社群里有很多非常好的程序，我们可以查看他的代码，社群里的程序是我们非常好的学习资源。

我们学到的知识点：

1，社群程序的搜索与下载

2，老外的翻译水平好差，服务器也差

3，社群里的程序是非常好的学习资源
